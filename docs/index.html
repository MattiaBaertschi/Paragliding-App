Digitales Logbuch

Das Paragliding-App bietet dir die Möglichkeit deine Flüge digital zu verwalten. Mühsame Einträge in ein analoges Logbuch, das immer mal gerne verloren gehen kann, bleiben dir so erspart

Call to action

IGC-Upload

Anstatt die Angaben zum Flug mühsam aufzuschreiben, kanst du ganz einfach das IGC-File deines Bordcomputers in die Anwendung einlesen. So hast du einen neuen Flug binnen Sekunden dokumentiert. Ausserdem werden alle Originaldaten in einer Datenbank abelegt uns sicher verwahrt.
